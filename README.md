## 📌 Sobre o Projeto

O **SaúdeTech** é uma solução desenvolvida para modernizar o processo de gestão pública em saúde no município de Londrina.

A plataforma utiliza tecnologia para automatizar a **triagem documental**, reduzindo falhas manuais, retrabalho e o tempo de análise dos processos da Vigilância Sanitária.

---

## 🎯 Objetivo

Melhorar a eficiência da gestão pública em saúde por meio da automação de processos, proporcionando:

- Mais agilidade  
- Redução de erros  
- Melhor organização das informações  
- Apoio à tomada de decisão  

---

## ⚙️ Funcionalidades

- Upload de documentos  
- Validação automática com OCR  
- Checklist de documentos obrigatórios  
- Classificação de risco com IA  
- Dashboard para análise de processos  

---

## 🧠 Tecnologias

- HTML, CSS e JavaScript  
- OCR (Reconhecimento de Texto)  
- Inteligência Artificial (conceitual)  
- Integração com sistema SEI  

---

## 📊 Benefícios Esperados

- Redução do tempo de análise  
- Diminuição do retrabalho  
- Padronização dos processos  
- Melhoria no atendimento à população
